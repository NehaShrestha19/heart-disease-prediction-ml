# heart-disease-prediction-ml
Machine learning project to predict heart disease using Logistic Regression and Random Forest
# Heart Disease Prediction using Machine Learning

## Overview
This project uses machine learning techniques to predict whether a patient has heart disease based on medical data.

## Models Used
- Logistic Regression
- Random Forest
- Baseline (Majority Class)

## Results
- Logistic Regression Accuracy: 95%
- Random Forest Accuracy: 88.33%
- Baseline Accuracy: 60%
- Cross-validation Accuracy: ~83%

## Key Features
- Exploratory Data Analysis (EDA)
- Correlation heatmap
- Confusion matrix
- ROC curve
- Model comparison

## Tools & Libraries
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Dataset
UCI Heart Disease Dataset

1. Open the notebook
2. Run all cells
